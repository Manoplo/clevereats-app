# Fuentes Space Grotesk

Esta carpeta contiene los archivos de fuente Space Grotesk para la aplicación.

## Archivos requeridos

Para que las fuentes funcionen correctamente, necesitas agregar los siguientes archivos en esta carpeta:

- `SpaceGrotesk-Regular.ttf` (o `.otf`)
- `SpaceGrotesk-Medium.ttf` (o `.otf`)
- `SpaceGrotesk-Bold.ttf` (o `.otf`)

## Cómo obtener las fuentes

1. Descarga Space Grotesk desde [Google Fonts](https://fonts.google.com/specimen/Space+Grotesk)
2. Extrae los archivos de fuente
3. Copia los archivos `.ttf` o `.otf` a esta carpeta con los nombres exactos:
   - `SpaceGrotesk-Regular.ttf`
   - `SpaceGrotesk-Medium.ttf`
   - `SpaceGrotesk-Bold.ttf`

## Nota importante

Los nombres de los archivos deben coincidir exactamente con los nombres definidos en `constants/fonts.ts`. Si los archivos tienen nombres diferentes, actualiza el archivo de configuración de fuentes.
